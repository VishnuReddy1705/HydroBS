# HydroBS Project Context

## Project Name
HydroBS - Smart Water Utility Management & Billing Platform

---

## Purpose

HydroBS is a full-stack SaaS web application for apartment communities to monitor water consumption, generate bills, manage residents, track purchased water, visualize analytics, and improve water conservation.

---

## Technology Stack

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Hibernate/JPA
- Maven

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- Recharts

### Database
- PostgreSQL
