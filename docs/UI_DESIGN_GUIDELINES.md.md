# HydroBS UI Design Guidelines

---

# Design Philosophy

HydroBS should look like a premium enterprise SaaS platform.

Every page should feel like it belongs to the same application.

Never design pages independently.

The Dashboard is the reference design for the entire project.

Every future page must match the Dashboard's design language.
